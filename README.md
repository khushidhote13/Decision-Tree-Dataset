# Titanic Survival Prediction using Decision Tree

## Project Overview

This project uses the Titanic dataset to predict passenger survival using the Decision Tree Machine Learning algorithm. The dataset was cleaned, analyzed, and used to train a classification model.

## Dataset

* Dataset: Titanic Dataset
* Source: Kaggle Titanic Dataset
* Target Variable: Survived

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook

## Project Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Model Building using Decision Tree Classifier
6. Model Evaluation
7. Result Interpretation

## Model

The Decision Tree Classifier was trained to predict whether a passenger survived the Titanic disaster based on available features.

## Results

The model was evaluated using standard classification metrics and achieved satisfactory prediction performance on the dataset.
